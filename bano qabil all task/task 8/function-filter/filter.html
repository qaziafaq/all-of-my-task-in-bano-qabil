Level 2: Array.filter()

Filter out the weak and keep only the champions that pass the test.

Syntax

array.filter((item, index, array) => {
  return condition;
});

Purpose

Creates a new array containing only the items that satisfy the callback condition.

Example

const numbers = [1, 2, 3, 4, 5, 6];
const evens = numbers.filter(num => num % 2 === 0);
console.log(evens); // [2, 4, 6]

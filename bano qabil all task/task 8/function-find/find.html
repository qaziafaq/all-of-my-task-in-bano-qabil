Level 5: Array.find()

Target the first winning item that meets the condition.

Syntax

array.find((item, index, array) => {
  return condition;
});

Purpose

Returns the first element that satisfies the callback, or undefined if none match.

Example

const users = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
  { id: 3, name: 'Charlie' }
];
const user = users.find(u => u.id === 2);
console.log(user); // { id: 2, name: 'Bob' }

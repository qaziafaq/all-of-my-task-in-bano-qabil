Level 4: Array.forEach()

Activate every item in the squad and perform an action for each one.

Syntax

array.forEach((item, index, array) => {
  // do something with item
});

Purpose

Runs a callback for each array element without producing a new array.

Example

const fruits = ['apple', 'banana', 'cherry'];
fruits.forEach((fruit, index) => {
  console.log(index + ': ' + fruit);
});

Level 1: Array.map()

Use map() to power-up each item in your array and build a new transformed squad.

Syntax

array.map((item, index, array) => {
  return transformedItem;
});

Purpose

Transforms every element in an array into a new form without changing the original array.

Example

const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8]

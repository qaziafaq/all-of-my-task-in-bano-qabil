Level 3: Array.reduce()

Combine all elements into one final score using reduce().

Syntax

array.reduce((acc, item, index, array) => {
  return updatedAcc;
}, initialValue);

Purpose

Reduces an array to a single value, like a score total or summary object.

Example

const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((acc, num) => acc + num, 0);
console.log(sum); // 10
